# Datenschutzerklärung — Side A

**Stand:** Juni 2026  
**Entwickler:** Sight-A  
**Kontakt:** sight.a@icloud.com

---

## 1. Grundsatz

Side A speichert alle Daten ausschließlich lokal auf deinem Gerät. Es werden keine persönlichen Daten an uns oder Dritte übertragen.

---

## 2. Erhobene Daten

Side A erhebt und speichert keine personenbezogenen Daten. Alle Inhalte (Alben, Tracks, Kategorien, Cover-Fotos, Notizen) werden ausschließlich in der lokalen Datenbank auf deinem iPhone gespeichert.

---

## 3. Drittanbieter

### MusicBrainz
Beim Suchen von Alben per Barcode oder Katalognummer werden Anfragen an die MusicBrainz API gesendet. Dabei wird lediglich der Suchbegriff (Barcode oder Katalognummer) übertragen. Es werden keine personenbezogenen Daten übermittelt.  
Datenschutzerklärung: https://metabrainz.org/privacy

### Discogs
Wenn du in den Einstellungen ein persönliches Discogs-Token hinterlegst, werden Suchanfragen an die Discogs API gesendet. Das Token wird ausschließlich lokal auf deinem Gerät gespeichert. Es werden keine weiteren personenbezogenen Daten übertragen.  
Datenschutzerklärung: https://www.discogs.com/privacy

### Apple Kontakte
Beim Erfassen verliehener Alben kannst du optional einen Kontakt aus deinem Adressbuch auswählen. Der Zugriff erfolgt über den systemseitigen Kontakt-Picker von Apple — Side A erhält dabei keinen direkten Zugriff auf dein Adressbuch. Es wird lediglich der ausgewählte Name lokal gespeichert.

---

## 4. Backup & Export

Die Export-Funktion erstellt eine JSON-Datei mit deinen Albumdaten. Diese Datei verlässt dein Gerät nur, wenn du sie aktiv über die Teilen-Funktion weiterleitest. Side A hat keinen Zugriff auf exportierte Dateien.
